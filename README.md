# Ucapan Ulang Tahun Tesa

Website ucapan ulang tahun sederhana untuk Tesa. Project ini dibuat dengan HTML, CSS, dan JavaScript vanilla, tanpa lagu dan tanpa foto.

## Struktur Folder

```text
ucapan-ulang-tahun-tesa/
|-- index.html
|-- style.css
|-- script.js
|-- README.md
|-- vercel.json
`-- assets/
```

## Cara Menjalankan di Local

1. Buka folder project.
2. Buka `index.html` di browser.

## Cara Deploy ke Vercel

1. Upload project ke GitHub.
2. Login ke Vercel.
3. Klik Add New Project.
4. Import repository GitHub.
5. Framework preset pilih Other.
6. Build Command dikosongkan.
7. Output Directory dikosongkan atau gunakan root project.
8. Klik Deploy.
