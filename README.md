# Proposal Greeting Kak Nadia

Website ucapan sederhana untuk merayakan Kak Nadia yang sudah menyelesaikan ujian proposal. Project ini dibuat dengan HTML, CSS, dan JavaScript vanilla sehingga ringan, responsif, dan siap dideploy sebagai static website di Vercel.

## Struktur Folder

```text
proposal-greeting-kak-nadia/
|-- index.html
|-- style.css
|-- script.js
|-- README.md
|-- vercel.json
`-- assets/
    |-- foto.jpg
    `-- lagu.mp3
```

## Cara Menjalankan di Local

1. Buka folder project.
2. Pastikan foto ada di `assets/foto.jpg`.
3. Pastikan lagu ada di `assets/lagu.mp3`.
4. Buka `index.html` di browser.

## Cara Deploy ke Vercel

1. Upload project ke GitHub.
2. Login ke Vercel.
3. Klik Add New Project.
4. Import repository GitHub.
5. Framework preset pilih Other.
6. Build Command dikosongkan.
7. Output Directory dikosongkan atau gunakan root project.
8. Klik Deploy.

## Catatan

Jika lagu tidak terdengar, pastikan file `assets/lagu.mp3` benar-benar ada dan nama filenya sama persis.
